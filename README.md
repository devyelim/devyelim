# Êmily Araújo

**`Desenvolvedora Web`**

Me chamo Emily Araújo e sou estudante de **Análise e Desenvolvimento de Sistemas** pelo Instituto Infnet de Tecnologia, no Rio de Janeiro. Tenho foco em desenvolvimento de sistemas web, com maior interesse em back-end, organização de regras de negócio e construção de aplicações bem estruturadas.
